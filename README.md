# (WIP) Simple-CRUD-Application
Made with:
- Angular 19
- Bootstrap 5
- Go 1.24

## How to Run
### Prerequisites

Make sure you have the following installed on your system:

- [Docker](https://www.docker.com)

### Command
Run `docker compose up -d`.
